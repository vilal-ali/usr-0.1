About Us

NLP Project

Universal Semantic Representation (USR) captures the meaning expressed by a sentence in a discourse. Any sentence in any language can be represented via its USR which makes it more meaningful. It helps the user in understanding the grammatical connections better, which further translates to better language learning experience.

Our project is a web application that allows users to add and import any text and generate its USR. Every sentence is divided into word chunks which are editable. Inter-sentence and Intra-sentence connections are also shown for the benefit of the user.
